# BharatMart Sales Analysis Dashboard (Power BI)

 Project Overview
This Power BI dashboard provides a comprehensive analysis of BharatMart's sales performance across states, cities, product categories, and customer segments.

It enables stakeholders to monitor key performance indicators and make data-driven business decisions.

##  Key Metrics
- Overall Sales Amount
- Quantity Sold
- Total Transactions
- Total Customers
- Revenue per Customer


## Dashboard Insights
- State-wise and city-wise sales performance
- Category-wise quantity sold analysis
- Tier-wise revenue contribution
- Supplier-wise quantity distribution
- Customer membership segmentation


##  Tools & Technologies
- Power BI
- DAX
- Excel
